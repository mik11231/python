# Advent of Code 2022

This folder follows the same day-based layout as `advent2018`, `advent2019`, and `advent2020`.

## Structure

- `DayN/dN_input.txt`: puzzle input
- `DayN/dN_instructions.html`: puzzle instructions page (saved HTML)
- `DayN/dayN.py`: Part 1 solution
- `DayN/dayN_part2.py`: Part 2 solution
- `DayN/test_example.py`: AoC example checks (optional)
- `legacy/`: original scripts retained for reference

All listed answers were submitted and accepted for Advent of Code 2022.
